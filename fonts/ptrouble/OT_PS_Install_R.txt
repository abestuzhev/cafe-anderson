Установка шрифтов OpenType PS.

- Шрифты OpenType на основе PostScript (OT/PS) состоят из одного файла с расширением OTF. 
- Шрифты OpenType могут быть установлены на любой версии Windows, а также на Mac OS 7.x-10.x.
- Шрифты OpenType включают в себя кодировку Unicode, которая поддерживает практически все мировые языки.
Это важное преимущество формата TrueType теперь совмещено с данными в формате PostScript.
- Шрифты OpenType можно инсталлировать средствами операционной системы или с помощью шрифтовой утилиты Adobe Type Manager.


Инструкции по установке шрифтов в различных ОС:

- - - - - -
Windows 95/98/NT/ME

Для того, чтобы использовать OpenType PS шрифты в Windows 95/98/NT/ME, вы должны предварительно установить
Adobe Type Manager (ATM) версии 4.1.2 или новее и установить шрифты с помощью ATM.
Вы можете скачать свежую версию ATM Light бесплатно с сайта Adobe http://www.adobe.com/products/atmlight/main.html 

Установка OT шрифтов в Windows 95/98/ME и Windows NT 4.0 с использованием ATM Light 4:
1. Выберите Пуск > Программы > Adobe > Adobe Type Manager. Если вы установили ATM в другую программную группу, выберите её.
2. В программе ATM, выберите закладку Fonts.
3. Выберите Browse For Fonts во всплывающем меню Source. 
4. Выберите папку в которой лежат шрифты, которые вы хотите установить (Проверьте, распаковали ли вы все архивы). 
5. Выберите шрифты, которые хотите установить.  
6. Нажмите кнопку Add (Добавить) для окончания установки.
7. Выйдите из ATM. 

После инсталляции, если вы хотите, чтобы обновился список шрифтов в уже запущенных приложениях, вам нужно будет
перезапустить их.

- - - - - -
Windows 2000/XP

Windows 2000 and Windows XP (Home and Professional) имеют встроенную поддержку OT шрифтов. 

1. Откройте Панель управления (Control Panel). В Windows 2000 дважды кликните на My Computer, затем дважды щёлкните
на иконку Control Panel. В Windows XP нажмите на кнопку Start и затем на Control Panel.
2. Дважды щёлкните на иконку Fonts в окне Control Panel. 
3. В окне Fonts выберите File -> Install New Font. 
4. Укажите папку, где лежат шрифты, которые бы Вы хотели установить (Проверьте, распаковали ли Вы предварительно архивы). 
5. Выберите шрифты, которые Вы хотите установить. Можете указать один шрифт или с помощью удержания клавиши Ctrl и 
щелчка мыши по шрифту выбрать несколько. 
6. Нажмите OK.  
7. Закройте окно Fonts. 

После инсталляции, если вы хотите, чтобы обновился список шрифтов в уже запущенных приложениях, вам нужно будет
перезапустить их.

- - - - - -
Mac OS 7.x-9.x

Для шрифтов OpenType PS (OT/PS) во всех версиях Mac OS, за исключением версии Mac OS X, необходим Adobe Type Manager (ATM),
чтобы шрифты отображались более гладко на экране и при выводе на печать.
Для установки OT/PS шрифтов нужен ATM версии 4.6.2 или новее. 
Вы можете скачать свежую версию ATM Light бесплатно с сайта Adobe http://www.adobe.com/products/atmlight/main.html 

- - - - - -
Mac OS X

Закройте все открытые приложения перед установкой шрифтов. 
Новые шрифты не появятся в некоторых приложениях, если эти приложения будут запущены в процессе установки шрифта. 

1. Откройте папку со шрифтами, которые вы хотите установить. 
2. Выберите файлы-чемоданы шрифтов, которые вы хотите установить. 
3. Удостоверьтесь, что вы выбрали шрифты, а не папки, содержащие шрифты.
3. Перетащите выбранные файлы в папку Fonts внутри папки Library на диске Mac OS X.
